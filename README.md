# üß© Optimization of FDM Process for Improved Print Quality

This project focuses on optimizing the **Fused Deposition Modeling (FDM)** 3D printing process to achieve improved print quality. It involves analyzing key process parameters such as **layer thickness, print speed, nozzle temperature, and infill density** using experimental design and statistical optimization techniques.

The goal is to minimize **surface roughness**, improve **dimensional accuracy**, and enhance the **mechanical strength** of printed parts.

- Designed 3D models using **CAD software**
- Conducted printing experiments using **FDM 3D printer**
- Analyzed data using **statistical optimization tools**
- Evaluated surface finish and part strength
